# 📚 Vocabulary Bot

Telegram-бот для изучения и запоминания новых слов.

## 🚀 Быстрый старт

### 1. Создайте бота в Telegram

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Придумайте имя и username для бота
4. Скопируйте полученный токен

### 2. Настройте проект

```bash
# Перейдите в папку проекта
cd Vocabulary_Bot

# Создайте виртуальное окружение (рекомендуется)
python -m venv venv
source venv/bin/activate  # для macOS/Linux

# Установите зависимости
pip install -r requirements.txt

# Создайте файл .env с токеном
cp .env.example .env
# Откройте .env и вставьте свой токен
```

### 3. Запустите бота

```bash
python bot.py
```

## 📱 Функции бота

| Команда | Описание |
|---------|----------|
| `/start` | Начать работу с ботом |
| `/add` | Добавить новое слово |
| `/words` | Показать мои слова |
| `/train` | Начать тренировку |
| `/stats` | Посмотреть статистику |
| `/help` | Помощь |

## 🛠 Технологии

- Python 3.10+
- python-telegram-bot 21.0
- SQLite (база данных)
- python-dotenv (конфигурация)

## 📝 Как добавить слово

1. Нажмите "➕ Добавить слово" или отправьте `/add`
2. Введите слово на иностранном языке
3. Введите перевод
4. Готово! Слово сохранено в вашу коллекцию

## 🎯 Режим тренировки

1. Нажмите "🎯 Тренировка" или отправьте `/train`
2. Бот покажет слово — постарайтесь вспомнить перевод
3. Нажмите "Показать ответ" для проверки
4. Отметьте, знали вы слово или нет
5. Бот адаптируется и чаще показывает сложные слова
